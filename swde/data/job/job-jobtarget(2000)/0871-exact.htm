												<p>Exalead (a Dassault Syst&egrave;mes company) is seeking an experienced and talented individual to fill a Technical Writer (CDI) position in Paris.</p> <p>The successful candidate must have more than five years experience as an English mother tongue technical writer. You will be in charge of the following duties:</p> <ul> <li> Establish documentation needs and obtain product knowledge with the developers</li> <li> Write administration, user, programming and installation guides, by interviewing developers and by using the product; includes the creation of text (following Information Mapping techniques) and schemas. </li> <li> Write technical white papers</li> <li> Generate online help with ePublisher</li> <li> Create flash tutorials for the product and SDKs using BB Flashback - screen recorder</li> <li> Follow and maintain documentation requests (tickets) in internal tracking system</li> <li> Deliver documentation in version control system and in Content Management System (Joomla); plus other Joomla administration tasks</li> </ul>
											
										
										
										
											<br><b>Job Requirements</b><br>
											
												<ul> <li> Information Mapping</li> <li> Adobe FrameMaker</li> <li> Webworks ePublisher Pro and Express</li> <li> Tortoise SVN version control</li> <li> Adobe Acrobat &amp; Photoshop</li> <li> Microsoft Office: Word, PowerPoint, Outlook</li> <li> English mother tongue with excellent spoken French. Ability to write French a plus</li> <li> O/S: Windows 7 and Linux are recommended</li> </ul> <h4>About Exalead:</h4> <p>Exalead&reg; is a global software provider in the enterprise and Web search markets, and the maker of CloudView, the industry's top platform for Search-Based Applications (SBAs).
