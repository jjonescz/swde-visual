About the Job

NOSC Security Analyst-8 positions 

Boulder, CO

All Shifts still available (specifically targeting third shift and weekends)

Start Date: Dec 1st, long term several years+

Must be a US Citizen (secret clearance or TS preferred but not mandatory as we can process for Interim clearance to start)

Job Description: 

Position is for a security and network analyst working as a member of a managed services Network Operations and Security Center (NOSC.) The focus of the NOSC analyst will be to monitor and respond to security, network and service monitoring events generated by a variety of applications including, but not limited to:

 

·        McAfee Total Protection Suite

·        IBM SiteProtector/Proventia HIDS

·        Tivoli Security Operations Manager

·        Tenable Security Center

·        Tenable Nessus

·        IBM SiteProtector/Proventia NIDS

·        Cisco Security Manager

·        Cisco Secure ACS

·        Juniper Network Security Manager

·        Checkpoint Smart-1

·        EMC Ionix Network Configuration Manager (Voyence)

·        Tivoli Network Manager

 

NOSC analysts are required to understand and apply knowledge of the associated tools when assisting teams and customers, but may not know all the listed applications, or may have experience with other comparable applications.  Position will require day-to-day monitoring of the enterprise security solutions as well as assisting team members with additional system and network security applications. Responsibilities will be to monitor all aspects of the solution, to include on-going documentation, maintenance, training of operations staff, and continuing optimization of the suite to provide maximum security protection and value. May be engaged as a resource who advises others on tools, technology, and methodology related to implementing IT solutions. Within the defined specialty, may be required to complete technical tasks for the project or new project proposals.

 

Required Skills: 

Must demonstrate analytical ability in developing and integrating solution deliverables and use judgment to resolve problems related to the technical solutions being implemented. 

 

Uses discretion and has latitude to improve the technical procedures and processes within area of technical specialty to enhance the implementation of a customer solution.

 

Desired Experience: 

More than 3 years of experience related to the following specialized technical skills:

·        Monitoring and maintaining system, network and security solutions

·        Enterprise monitoring and improvement of host-based intrusion prevention, security information and event management, firewall, network intrusion detection systems, vulnerability scanners and network management.

·        In-depth understand of ports, protocols, system hardening and system functions (file I/O, registry, file permissions) analysis as it relates to network security

·        Experience using troubleshooting techniques including; system and network logs, network sniffers, syslog, Microsoft’s Sysinternals

·        Understanding of information security principles as it relates to system and network security.

·        Ability to create formal documentation for systems administration, operations, and maintenance.

·        Understanding of formal processes for change and release management

Excellent teamwork, verbal and written communications skills

 

Technical Skills: 

Desired Experience:  More than 3 years of experience related to the following:

·        Security and network incident response

·        System and network forensics as it relates to incident response

·        UNIX/Linux system administration skills

·        Windows system administration skills

·        Network administration skills

·        Firewall, IDS, IPS, proxies and content filter administration skills.

·        Desirable to have Perl scripting skills capable of text parsing, manipulation, and output

·        Security operations and/or engineering experience

·        Security incident response experience

·        Industry security and systems certifications (CISSP, CCSP, CCNP, CCNA, SSCP, GCFW, GCIA)

·        ITIL Certification

·        Application Software:  McAfee Total Protection Suite, IBM SiteProtector, IBM Proventia HIDS, Tivoli Security Operations Manager, Tenable Security Center, Tenable Nessus, IBM Proventia NIDS, Cisco Security Manager, Cisco Secure ACS, Juniper Network Security Manager, Checkpoint Smart-1, EMC Ionix Network Configuration Manager (Voyence) and Tivoli Network Manager

Diagnostic Software:  WireShark, Windows Command-line, UNIX/Linux Command-line, Process Explorer, NetMon, Syslog, tcpdump

 

Primary Job Duties: 

Job Duty 1 with 50% of time – Monitor enterprise security solutions

Job Duty 2 with 20% of time – Provide documented analysis of and make determinations about security events

Job Duty 3 with 10% of time – Escalate security events per NOSC procedures

Job Duty 4 with 10% of time – Participate in security incident response activities

Job Duty 5 with 10% of time – Assist in the development and maintenance of NOSC operations procedures

 

PLEASE NOTE: Candidates submitted must be US Citizens able to obtain/pass FULL Background Investigation
